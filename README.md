# radbird

radbird is a tool to help setting up a Twitter bot publishing radiation measurement readings. It is mainly developped by [Antonin Segault](http://retrodev.net), a Ph.D student in the [ELLIADD laboratory](http://elliadd.univ-fcomte.fr) of the [Franche-Comt√© University](http://univ-fcomte.fr).

radbird requires the [Interfascia](https://github.com/brendanberg/interfascia) native Processing library and the [Twitter4J](http://twitter4j.org/en/index.html) java library.
